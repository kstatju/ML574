%matplotlib notebook

Let layer 1 refer to the input layer, layer 2 to the hidden layer, and layer 3 to the output layer.

For each layer, number nodes beginning with 1. Since we have three classes, the output layer has 3 nodes.

Then for the edge from node 1 in layer 2 to node 3 in the output layer, set

clf.coefs[1][0][2] = some number

the first index is the layer the edge originates from. The next two indices are for the node number in the left layer and right layer respectively. Recall Python uses 0 to refer to the first index.

w11 = np.linspace(-10, 10, 75)
w21 = np.linspace(-10, 10, 75)

# create a meshgrid and evaluate training MSE
W11, W21 = np.meshgrid(w11, w21)
MSEmesh = []
for coef1, coef2 in np.c_[W11.ravel(), W21.ravel()]:
    clf.coefs_[1][0][2] = coef1 
    clf.coefs_[0][0][1] = coef2
    MSEmesh.append( [clf.score(Xtrain,ytrain)] )

MSEmesh = np.array(MSEmesh)

# Put the result into a color plot
MSEmesh = MSEmesh.reshape(W11.shape)

ax = plt.axes(projection='3d')
ax.plot_surface(W11, W21, MSEmesh, rstride=1, cstride=1,
                cmap='viridis', edgecolor='none')
